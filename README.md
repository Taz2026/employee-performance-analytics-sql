# employee-performance-analytics-sql
Business Problem & Context

In supply chain operations, employee performance directly influences service levels, cost efficiency, and operational reliability. However, performance data is often disconnected from measurable business KPIs, making appraisals and promotion decisions subjective and inconsistent.

This project designs a structured SQL database that centralizes employee performance data, links it to key supply chain KPIs, and generates appraisal-ready reports. By enabling weighted performance scoring and measurable evaluation criteria, the system supports transparent, data-driven promotion decisions and aligns workforce performance with business objectives.
